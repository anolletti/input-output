How-To
Mode d'emploi
  1. 1. Enter an input value.
  2. 1. Entrez un nombre d'entrée.
  3. 2. Click on the robot or ENTER.
  4. 2. Cliquez sur le robot ou peser la touche Entrée.
  5. 3. The robot will transform your number into a new number. This is called the output value.
  6. 3. Le robot transforme les nombres d'entrées. Ce nombre s'appelle le nombre de sortie.
  7. 4. Continue entering input values until you know the robot's transformation rule.
  8. 4. Entrez plus de nombre d'entrée jusqu'à ce que vous ayez déterminer la règle d'entrée et sortie du robot.
